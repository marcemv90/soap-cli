# Command-line usage

This section explains how to use `SoapCli` from the command line, including:

- Basic invocation and providing the SOAP request body.
- Sending attachments (inline Base64 and MTOM/XOP).
- Authenticating with client certificates and HTTP Basic.
- Using profiles to avoid repeating configuration.
- Working behind HTTP/HTTPS and SOCKS proxies.
- Choosing color themes.
- Understanding return codes and errors when scripting.
